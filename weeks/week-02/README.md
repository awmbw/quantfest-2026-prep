### 🔑 Key Formulas to Memorize

```
P(A|B) = P(B|A) · P(A) / P(B)           # Bayes' theorem
E[X + Y] = E[X] + E[Y]                   # Linearity (always true!)
E[XY] = E[X]·E[Y]                        # Only if independent
Var(X) = E[X²] - (E[X])²                 # Variance shortcut
Geometric: E[X] = 1/p                    # Trials until first success
```
