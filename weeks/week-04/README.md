### 🔑 Key Formulas

```
C(n,k) = n! / (k!(n-k)!)                    # Combinations
P(n,k) = n! / (n-k)!                         # Permutations
Stars & Bars: C(n+k-1, k-1)                  # n items, k bins
Inclusion-Exclusion: |A∪B| = |A|+|B|-|A∩B|   # Extends to n sets
Catalan: C_n = C(2n,n)/(n+1)                 # Many combinatorial structures
Derangements: D_n = n! · Σ(-1)^k / k!        # k from 0 to n
```
