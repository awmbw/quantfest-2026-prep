- Start a personal "techniques notebook" in `notes/`.

## 💻 Coding — YouKn0wWho Academy

### Target: Probabilities & Expected Values — Part 1
- [Topic List Link](https://youkn0wwho.academy/topic-list)
- Focus on problems rated ≤ 2200
- Aim to complete ~5–7 problems from this module

### Approach
1. Read the editorial/tutorial for the topic first
2. Attempt problems on your own for 30–45 min
3. If stuck, read hints (not full solution)
4. After solving, read the editorial to compare approaches
5. Save your solutions in `solutions/coding/topic-list/`
