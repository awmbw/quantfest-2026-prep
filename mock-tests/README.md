# 🧪 Mock Tests

## Format

Each mock test simulates the Quantfest prelims:
- **12 problems** (5 math/probability + 7 coding)
- **3 hours** strict time limit
- **No external resources** (no looking up formulas, no AI assistance)

## How to Construct a Mock Test

### Math Problems (pick 5)
- Choose from unsolved or long-ago-solved Green Book problems
- Mix topics: 2 probability, 1 combinatorics, 1 expected value, 1 statistics/misc

### Coding Problems (pick 7)
From Codeforces, filter by relevant tags:
- 2 problems rated 800–1200 (warm-up)
- 3 problems rated 1200–1600 (core)
- 2 problems rated 1600–2000 (hard)
- Use tags: probabilities, combinatorics, dp, math
