## 📌 What is Quantfest?

Quantfest is a **national-level quantitative finance competition** organized by IICPC (InterCollegiate Informatic and Competitive Programming Camp). It simulates real-world quantitative challenges across algorithmic trading, probability, market microstructure, and quantitative research.

### Competition Format (based on 2025)

| Stage | Format | Details |
|-------|--------|---------|
| **Prelims** | Online / On-campus | 12 problems: 5 math/probability + 7 coding (on Codeforces) |
| **Regionals** | In-person at IIT campuses | Heuristic simulations & quant problem sets |
| **Finals** | 3-day event at IIT Madras | Market simulations, trading games, advanced challenges |

## 🎯 Preparation Strategy (11-Week Overview)

| Phase | Weeks | Focus | Goal |
|-------|-------|-------|------|
| **Phase 1: Foundations** | 1–3 | Probability basics, Green Book Ch. 1–3, DP fundamentals | Build core math & coding intuition |
| **Phase 2: Deep Dive** | 4–6 | Advanced probability, combinatorics, expected value problems | Tackle competition-level problems |
| **Phase 3: Integration** | 7–9 | Mixed problem sets, Codeforces contests, speed drills | Build speed & accuracy under pressure |
| **Phase 4: Peak & Taper** | 10–11 | Full mock tests, weak-area review, mental preparation | Peak performance on competition day |

## 🚀 Quick Start

1. **Clone this repo** and start with [Week 1](weeks/week-01/README.md)
2. **Track your progress** in [PROGRESS.md](PROGRESS.md) — check off items daily
3. **Store solutions** in the `solutions/` directory, organized by type
4. **Take notes** in `notes/` — especially for tricky concepts you want to revisit
5. **Do mock tests** weekly starting Week 7 in `mock-tests/`

## 📊 Daily Time Commitment

| Activity | Time | Notes |
|----------|------|-------|
| Green Book problems | 45 min | 3 problems/day, review solutions |

| Puzzle solving | 20–30 min | Brainstellar, Jane Street, etc. |
| Notes & review | 15 min | Document learnings, review weak areas |
| **Total** | **~2.5–3 hrs/day** | Adjust based on your schedule |
